Tutorial for

Memory game by Mauro and James

We gaan een memory game bouwen
Door kaartjes om te draaien kunnen we de juiste combinaties vinden!

we gebruiken voor deze game 3 codetalen
  • HTML
  • CSS
  • JavaScript

Stap 1 - Werkstructuur klaarmaken

We gebruiken Visual Studio Code om onze code te schrijven

  • 1.1 - Startcode downloaden, en de zip uitpakken
  • 1.2 - Open de uitgepakte folder in Visual Studio Code
  • 1.3 - Installeer de live server extensie en start de live server recht onder in de blauwe balk

stap 2 - beginnen met coderen

Nu komt het echte werk. We gaan een beetje code schrijven zodat we al iets kunnen zien in onze browser.

  • 2.1 - Ga naar je index.html bestand, schrijf vervolgens in het bestand ! en druk op enter. Normaal krijg je dan je begin code.
  • 2.2 - Vervolgens schrijf je de onderstaande code tussen de body tags van je startcode "< body >...< /body >".

    • hier schrijven we eigenlijk onze html.
    We maken een div waar alles gaat inzitten de h1 op lijn 3 is onze titel. We hebben een div met class cards. Deze gaat later vervangen worden door onze javascript en hierin zullen ook onze kaartjes komen.
    De modal-overlay is het scherm dat we zullen krijgen als we alle kaartjes juist omgedraaid hebben.
    Op de onderste lijn linken we gewoon ons javascript bestand

  • 2.3 - Onderstaande stukje code kan je kopieeren en plakken in je main.css file

    • Dit is eigenlijk een klein stukje css met algemene code die we veel zullen gebruiken.

  • 2.4 - Dit stukje code mag gekopieerd worden en geplakt worden in het main.js bestand

    • Wat we hier eigenlijk doen is het maken van een array met als naan "persons" met hierin allemaal verschillende objecten waarbij elk object een naam en een image bevat. Dit is de image die zal te zien zijn bij het draaien van het kaartje. we dupliceren deze array omdat er dus 2x dezelfde foto in het spel moet zitten.

stap 3 - Nu kleine pauze .... BAAAA nee verder met coderen :))

tot nu toe hebben we in onze 3 bestanden dat we gaan gebruiken al "iets" van code geschreven. TOP!
Dan kunnen we nu verder.

  • 3.1 - Deze code mag gekopieerd worden en geplakt worden in het main.js bestand onder hetgene wat er al staat

    • In deze JavaScript code maken we onze constante aan genaamd app, hierin zal alle code komen om ons spel te laten werken.

    We maken hier onze variabelen aan en roepen onze functies op. Door de setTimout() zorgen we ervoor dat de kaartjes de eerste 3 seconden zichtbaar zijn.

    Ook zien we de html die zal gebruikt worden voor 1 kaartje.

  • 3.2 - Hieronder zoveel code denk je waarschijnlijk, kopieer deze in de main.css.

    • Deze code is nodig om heel ons spel er visueel mooi uit te laten zien. Je kan later zelf nog gaan spelen met alle kleurtjes enzo voort.

  • 3.3 - Bij deze het laatste stukje code voor onze game. Kopieer deze nog in het main.js bestand (onder het gene wat we al hebben)

    • Hier gaan we eigenlijk gaan zeggen wat er moet gebeuren als we op een kaartje drukken. Het moet ten eerste rotate en dan moet het gaan vergelijken met de andere kaartjes enzo voort. Allemaal functies die doorlopen worden van als we op het kaartje drukken.

Nu kan je de game naar je eigen goesting gaan designen of gaan veranderen. Hartelijk bedankt om onze game te maken en te spelen!!
